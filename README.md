# LandingPage-AgroWorldArgentina
Landing page institucional oficial de AgroWorld Argentina. Plataforma diseñada para conectar el sector agropecuario con soluciones digitales modernas. El proyecto está optimizado para la conversión de leads, velocidad de carga y una experiencia de usuario fluida.

Puedes ver el sitio en vivo aquí:

El proyecto fue desarrollado utilizando un stack moderno para garantizar escalabilidad y rendimiento:
- Frontend: React.js con Vite.
- Estilos: CSS3.
- Formulario: Integración con EmailJS para la gestión de contactos sin backend propio.

Características Principales
- Diseño Responsive: Adaptado para dispositivos móviles, tablets y escritorio.
- Formulario de Contacto: Validación en tiempo real.
